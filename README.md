# MyGovCare

A mobile application that helps users find the nearest clinic and hospital near them based on their current GPS position, solving the problem of locating emergency medical facilities quickly.

## About

MyGovCare is a group project (ICT602) designed to address the critical need for quick access to medical facilities during emergencies. The application provides a simple and intuitive way for users to locate the nearest clinics and hospitals relative to their current GPS position.

## Features

- **Interactive Map** - View nearest clinics and hospitals marked on an interactive map based on your current GPS position
- **Location Details** - Click on map markers to see detailed information about specific clinics and hospitals
- **Theme Customization** - Customize the app's appearance with different theme options
- **User Authentication** - Secure login and sign-up functionality for users
- **Location Reports** - Web-based dashboard to view location reports including:
  - Username of reporters
  - User agent information
  - Timestamp of reports
  - GPS coordinates (latitude and longitude)

## Tech Stack

- **Frontend**: Android Studio, Java
- **Backend**: REST API, PHP
- **Database**: XAMPP (Apache, MySQL)
